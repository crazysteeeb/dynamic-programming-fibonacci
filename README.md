# Dynamic Programming Fibonacci

A Python implementation of Fibonacci sequence generation using both dynamic programming and optimized iterative approaches.

## Overview

This project demonstrates multiple approaches to solving the Fibonacci sequence problem while comparing memory usage and algorithm efficiency.

The project includes:
- Dynamic programming implementation
- Optimized iterative implementation
- User input handling
- Sequence computation analysis

## Features

- Fibonacci sequence generation
- Dynamic programming approach
- Optimized constant-space implementation
- Interactive user input
- Algorithm comparison

## Technologies Used

- Python

## Concepts Demonstrated

- Dynamic programming
- Iterative optimization
- Algorithm efficiency
- Time and space complexity
- Problem decomposition

## How to Run

```bash
python fibonacci.py
```
